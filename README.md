# STALZONE Server Swapper

> 🔄 Скрипт быстрой смены региона серверов для STALZONE — переключайтесь между российскими и глобальными серверами.

## 📋 Описание

STALZONE использует две отдельные папки для определения серверов, к которым подключается игра:

- `bin_global` — Европейские/Глобальные серверы
- `bin_ru` — Российские серверы

Этот скрипт меняет папки местами, позволяя переключаться между российскими и глобальными серверами одним кликом. Никакого ручного переименования.

## 🚀 Установка и использование

### 1. Скачайте скрипт

Клонируйте репозиторий или скачайте `swap_servers.bat` напрямую:

```bash
git clone https://github.com/your-username/stalzone-server-swapper.git
```
### 2. Поместите скрипт в папку с игрой
Скопируйте swap_servers.bat в директорию игры STALZONE: C:\Program Files (x86)\Steam\steamapps\common\STALZONE\

### 3. Запустите перед началом игры
Дважды кликните по swap_servers.bat перед запуском STALZONE через Steam.

### 4. Запустите игру
После завершения обмена запустите STALZONE как обычно через Steam. Игра будет использовать серверы из активной папки.
